# workshop-text-editor

This repository is focused on exploring the concept of a text editor workshop.

## Features
- Explore various text editing techniques
- Learn about keyboard shortcuts and their benefits
- Understand text file formats

## Technical Stack
- JavaScript
- HTML
- CSS

## Usage
To get started, clone the repository and open the `index.html` file in your browser.
